dtorch
======

Description
-----------

This package is the main module of this project for machine learning.
It is basicly a replica of **pytorch**.
It offer the possibility to create tensor, have them store elements for gradient calcul and create ML models.

.. important::
    This project is currently in progress. It's undergoing majors changes and is subject to changements
