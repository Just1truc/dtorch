dtorchtree
==========

Description
-----------

Tree type models are available in this package.
It is possible to train a tree on specific data or just use a pretrained one.

.. tip::
   If you ever make a model that you are proud of, do not hesitate to send it to me via email so that I can make it disponible to everyone.

Datasets are also available here.
